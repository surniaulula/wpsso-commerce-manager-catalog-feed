== Description ==

<h3>WPSSO Core Required</h3>

WPSSO Facebook Catalog Feed XML (WPSSO FCF) is an add-on for the [WPSSO Core plugin](https://wordpress.org/plugins/wpsso/), which provides complete structured data for WordPress to present your content at its best on social sites and in search results â€“ no matter how URLs are shared, reshared, messaged, posted, embedded, or crawled.
